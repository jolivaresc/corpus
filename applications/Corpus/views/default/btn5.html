Proyecto Final de Procesamiento de Corpus Textuales y Orales


El proyecto consistía en recopilar y compilar 50 documentos de reseñas de películas para crear una aplicación que pudiera hacer análisis de textos.
Las fuentes consultadas fueron:
  • http://cinescopia.com/
  • http://www.sensacine.com/
  • http://entrelalecturayelcine.blogspot.mx/
  • http://www.labutaca.net/
Los tipos de análisis que realiza esta aplicación son:
  • Búsqueda tipo web.
  • Búsqueda de concordancia.
  • Colocacaciones.
  • Análisis de sentimientos.
Link del repositorio (GitHub):
  • Proyecto Final de Procesamiento de Corpus Textuales y Orales
Programas y software utilizado para la elaboración de este proyecto:
  • Python 2.7: https://www.python.org/
  • Freeling: http://nlp.lsi.upc.edu/freeling/node/1
  • Manatee: https://nlp.fi.muni.cz/trac/noske/wiki/Downloads
  • Web2py: http://web2py.com/
